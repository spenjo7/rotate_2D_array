1. PURPOSE:
This .js file simply takes a 2 dimensional array and rotates it so that the rows become columns and the columns become rows. 

2. EXAMPLE:

	[
		['dog','horse'],
		['cat','bat'],
		['whale','mouse']
	]

		-- becomes --
		
	[
		["dog", "cat", "whale"],
		["horse", "bat", "mouse"]
	]

3. Usage:

let rotatedArray = rotate_2d_array( array );
